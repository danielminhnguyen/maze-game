Welcome to Maze Generator Algorithm

Try to use Visualise and press Re-Maze to see the algorithm in action. Use slider to adjust the speed

Use Edit Mode to drag entrance and exit around

Use Keyboard to move character around to the Exit

We use randomized depth-first algorithm:

  1. Given a current cell as a parameter
  2. Mark the current cell as visited
  3. While the current cell has any unvisited neighbour cells
    1. Choose randomly one of the unvisited neighbours
    2. Remove the wall between the current cell and the chosen cell
    3. Invoke the routine recursively for a chosen cell
    4. Keep track of the path taken
  4. Reach a dead-end with no available unvisited neighbour
    1. Take step back until found cell with valid neighbour
  5. Continue until all the cell has been visited
